## Segnalazione vulnerabilità

Per segnalare vulnerabilità di sicurezza in questo repository, apri un issue con il tag `security` o invia una mail a security@example.com (sostituisci con il contatto del progetto).

Se la vulnerabilità è critica e preferisci non aprire un issue pubblico, crea un contatto privato seguendo le istruzioni su GitHub per la segnalazione responsabile.

Grazie per contribuire alla sicurezza del progetto.
